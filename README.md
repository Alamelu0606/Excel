Insights Covered

Heart issues by age
Age vs illness prevalence (diabetic, normal, prediabetic)
Cancer distribution among patients
Charges distribution across age groups
Lifestyle risks (smoking, obesity, overweight, etc.)

Tools Used

Microsoft Excel
Pivot tables
Slicers
Charts (Pie, Radar, Stacked Bar, Column Chart)
Conditional formatting

Purpose
To demonstrate how Excel can be used to build analytical dashboards that help identify healthcare trends and support data-driven decisions.
